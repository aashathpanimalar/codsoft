RESULT MANAGEMENT SYSTEM(2023)

Using HTML CSS JavaScript PHP MySQL, I created a result management system project. The project's first page has two fields: faculty login and student login. Clicking the faculty login button takes us directly to the faculty login page. The faculty's username and password are unique to them and can only be entered by the faculty. If the username and password match, the user will be directed to the student mark database, where they can enter the student's name, register number, department, eight subject grades, result, and GPA. Upon selecting the student login option, a page will load. When the student's input must match the register number and date of birth that the faculty entered. Every result value needs to be stored in the database. PHP will be used to retrieve the data from the database when the user enters a unique register number. When a student wants to compute his CGPA, they should use the CGPA calculator button located at the bottom of the student login page.

Random Number Guessing Game

Using HTML, CSS, and JavaScript, I made a game/challenge named "Guessing the Number Challenge". In this project, a random number is automatically generated by the system, and the user inputs a number between 1 and 100. If both numbers match, the game is completed. For example, if the computer generates the number 26 and the user enters 50, the system will indicate that the number is too high, implying that the generated number is below 50. Then entering the number 22, the computer will say that the entered number is too low, indicating that the number is greater than 22. This is the general concept of this game. After finding the correct random number, the system will display the attempts made by you. If you want to play the game again, there's a "Play Again" button. Clicking that button allows you to replay the game. Through this small project, I'm confident that I've learned CSS, HTML, and JavaScript well.
